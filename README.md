# 数据库课程设计（饭店点餐）

## 项目简介
数据库课程设计（饭店点餐系统）是课程设计大作业，包含了对8张表的增删改查。1.数据库SQL文件2.E-R图3.每张表每个字段的详细说明

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
